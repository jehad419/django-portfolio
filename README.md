Django Portfolio Website

A personal portfolio website built with Django to showcase my skills, projects, and journey as a Computer Science student and aspiring Software Engineer.

Features:
* Responsive design
* About Me section
* Skills showcase
* Project showcase
* Contact information
* Clean and modern UI

Tech Stack:
* Python
* Django
* HTML
* CSS
  

For Installation:
git clone https://github.com/jehad419/django-portfolio.git
cd django-portfolio
pip install -r requirements.txt
python manage.py runserver
